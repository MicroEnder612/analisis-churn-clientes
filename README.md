# Predicción de Churn de Clientes

## Objetivo
Predecir si un cliente cancelará su servicio, usando el dataset público
Telco Customer Churn de IBM (7,043 registros).

## Proceso
1. Limpieza y transformación de datos (Python: pandas, NumPy)
2. Codificación de variables categóricas
3. Modelo de clasificación con Random Forest (Scikit-Learn)
4. Análisis de importancia de variables
5. Visualización de resultados (Matplotlib, Seaborn)

## Resultados
- Exactitud aproximada: 78-82%
- Variables más relevantes: tipo de contrato, antigüedad, cargos mensuales, soporte técnico

## Herramientas
Python · pandas · NumPy · Scikit-Learn · Matplotlib · Seaborn

## Datos
El archivo `data/WA_Fn-UseC_-Telco-Customer-Churn.csv` contiene el dataset
público de IBM utilizado en este proyecto (7,043 registros).

Fuente original: [Telco Customer Churn - Kaggle](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)
